01. How it works

Three steps. That's it.

From install to coding from your phone in under 60 seconds.

Step 01

Install the daemon

$ npx devbox-daemon

One command. No config. Works on macOS, Linux, and Windows with Node.js 18+.

Step 02

Scan the QR code

Open the PocketDev app on your phone and scan. Paired instantly. No passwords.

Step 03

Ship from anywhere

You: Build a REST API with auth
Claude: Creating files...
+ src/routes/auth.ts

Send prompts, watch code generate live, see diffs with red/green — all from your phone.
